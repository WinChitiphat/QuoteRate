Refresh Model

How often values update

QuoteRate checks the live dashboard payload every 1 minute and also supports manual refresh from the main dashboard. If one market source is temporarily unavailable, the app falls back to the remaining healthy sources instead of blanking the whole page.

Sources

Where each number comes from

  • OANDA powers the live fiat exchange cards and the USD/THB reference flow.
  • Bitkub provides crypto-to-THB order book pricing for USDT/THB.
  • Coinbase provides crypto-to-USD order book pricing for USDT/USD.
